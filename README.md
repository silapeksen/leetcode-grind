# Daily LeetCode Challenge

Welcome to our shared tracking repository! This project was created by two software engineers to sharpen our algorithmic thinking, master data structures, and—most importantly—build an unshakeable habit of **daily coding consistency**.

Our goal is to solve at least one LeetCode problem every single day, keep each other accountable, and push our problem-solving boundaries! 🔥
Everone uploads their solutions under their own folder name. 

File name format: `sila_001_two_sum.py`, `ipek_042_trapping_rain_water.cpp` etc.
